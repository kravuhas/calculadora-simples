imple Calculator

🧮 A basic calculator project built with Python. This program performs simple arithmetic operations in a quick and easy way.

📌 Features
Addition
Subtraction
Multiplication
Division
User-friendly input (via terminal)
📂 File Description
calculator.py

Main script that handles:

User input
Performing calculations
Displaying results
⚙️ Requirements
Python 3 installed

(No external libraries required)

▶️ How to Run

Clone the repository:
git clone https://github.com/kravuhas/calculadora-simples.git

Navigate to the folder:

cd calculadora-somples

Run the program:

python calculator.py
